# CONTEXT —— 摸鱼助手 · 词汇表（Glossary）

> 本文件只定义领域语言（术语 → 含义），**不包含任何实现细节**。
> 实现决策请放在代码、README 或 `docs/adr/`。

## 术语

### 观察渠道（Observation Channel）

别人感知"你是否还在操作电脑"的途径。本项目**只**针对：

- **屏幕画面**——别人通过屏幕共享 / 投屏 / 肉眼偷瞄，看到的是你这块屏幕显示的**画面**（场景 A / D）。

**明确排除**（不在本项目能力范围内）：

- **操作系统级监控**——通过 OS API 统计鼠标/键盘空闲时长、或定时截图的监控软件（场景 B）。浏览器扩展无法影响。
- **网页在线状态**——某网页系统依据网页内事件/心跳判断"在线/活跃"（场景 C）。本项目不以此为目标。

### 活跃伪装（Presence Simulation）

本项目的核心目标：让**屏幕画面**持续呈现出"有人正在浏览网页"的样子，从而在**观察渠道 = 屏幕画面**时，让旁观者误以为你仍在操作。

注意：活跃伪装作用于**画面可见性**，而非操作系统的真实输入。

### 自动滚动（Auto-Scroll）

活跃伪装的主力手段：让当前网页画面持续地、平滑地上下滚动，使画面"看起来在动"。

### 拟人停顿（Human-like Pause）

自动滚动过程中插入的、时长随机的暂停，用来模拟真人阅读时的停顿，避免画面呈现出机械、匀速的滚动特征。

### 可信度目标 / 扫一眼可信（Believability Bar / Glance-Level）

活跃伪装要达到的可信程度被**刻意限定**为"**扫一眼可信**"：旁观者**瞥一眼**屏幕，看到画面在动、有停顿，便认为"还有人在操作"即可。

**明确不追求**"经得起长时间紧盯"——旁观者若持续盯屏，会发现是在**反复滚动同一页面**。为此本项目**不引入**自动点击、自动换页等高风险拟人手段。

### 真实光标（Real Cursor）

操作系统绘制的鼠标指针。**浏览器扩展无法移动它。** 因此在"观察渠道 = 屏幕画面"时，旁观者看到的真实光标始终静止——这是本项目的一条硬边界。
